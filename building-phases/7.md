## Phase 7: Cancellation & Compensation

**Acceptance Criteria:** An attendee can cancel an order up to 24 hours before the event. The seat is released, a refund is simulated, and the waitlist is triggered if applicable.

**Azure Resources to Add:**

- None

**What you build:**

- `DELETE /orders/{orderId}` — triggers Cancellation Orchestrator
- Cancellation Orchestrator: validate cutoff → simulate refund → release seat → update status → trigger waitlist if needed
